# 데이터 시드 스크립트

## 개요

이 디렉토리에는 DynamoDB 테이블에 샘플 데이터를 삽입하는 시드 스크립트가 포함되어 있습니다.

## 사용 방법

### 1. CDK 스택 배포

먼저 DynamoDB 테이블을 생성해야 합니다:

```bash
npm run cdk:deploy
```

### 2. 환경 변수 설정 (선택사항)

기본값을 사용하지 않으려면 환경 변수를 설정할 수 있습니다:

```bash
export TABLE_NAME=PartnerConfirmationTable
export AWS_REGION=ap-northeast-2
```

### 3. 시드 스크립트 실행

```bash
npm run seed
```

또는 직접 실행:

```bash
ts-node bin/seed-data.ts
```

## 샘플 데이터

스크립트는 다음 5개의 예약 데이터를 삽입합니다:

1. **RES001** - 김철수 (서울 헤어샵, 헤어컷)
2. **RES002** - 이영희 (강남 네일샵, 네일아트)
3. **RES003** - 박민수 (홍대 마사지샵, 전신 마사지)
4. **RES004** - 정수진 (서울 헤어샵, 펌)
5. **RES005** - 최동욱 (신촌 피부관리샵, 피부관리)

## 테스트용 전화번호

샘플 데이터에는 다음 전화번호가 포함되어 있습니다:

- 고객: +821012345678, +821023456789, +821034567890, +821045678901, +821056789012
- 파트너: +821087654321, +821098765432, +821009876543, +821010987654

## 주의사항

- 실제 SMS 테스트를 위해서는 유효한 전화번호로 변경해야 합니다
- 스크립트는 기존 데이터를 덮어쓰지 않고 새로 삽입합니다
- 동일한 예약번호로 재실행하면 데이터가 업데이트됩니다
